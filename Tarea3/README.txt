	El presente es una tarea que corresponde al tema de tiempos de
	ejecucion de los programas. Se crean 2 algoritmos, uno para
	un ordenamiento con un selection sort y otro para una busqueda
	con el binary search. Ambos se implementan en Python y C++ para
	ver su comportamiento y resultados en cuanto a tiempos de ejecu-
	cion.
